# bloknot-volgograd.ru
 https://bloknot-volgograd.ru/

Томита-парсер в данном проекте используется для поиска упоминаний значимых персон и достопримечательностей.

Для запуска необходим томита-парсер, установка которого описана тут: https://github.com/yandex/tomita-parser

Для запуска данного проекта(в ОС Linux), необходимо поместить папку проекта в директорию с фалом запуска: tomita-parser с указанием файла конфигурации проекта, имеющем расширение .proto
Например, файл запуска лежит по следующему пути: ~/tomita-parser/build/bin/, в эту директорию необходимо поместить папку данного проекта. Теперь, чтобы запустить проект, можно воспользоваться командой: ./tomita-parser ./bloknot-volgograd.ru/tomita-parser_bloknot-volgograd/config.proto

Входным файлом является файл input.txt, который содержит текст статей.

Выходными файлами являются: output.txt - содержит входной текст, в который вставлены пометки с выделенными фактами, pretty_output.html - является подобием output.txt, только факты помещены в отдельные таблицы
